# 钱是什么?
在公元2022年, 钱可以用来花, 也可以赚来

# 功能

1. 花钱
    车/房/淘宝网/黑市/全球购/搞项目
    放贷/撒币
2. 赚钱
    农民/工人/码农/科研狗/金融狗/管理层
    借钱/印钱

注: 只是个体的花钱方式, 不包括理论实体的花钱方式


## 如何修改外观
